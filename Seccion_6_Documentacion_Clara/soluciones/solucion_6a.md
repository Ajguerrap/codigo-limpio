# Solución sugerida - Ejercicio 6a

```python
def multiplicar(x, y):
    """
    Multiplica dos números.

    Args:
        x (int or float): Primer factor.
        y (int or float): Segundo factor.

    Returns:
        int or float: El producto de x e y.
    """
    resultado = x * y
    return resultado
```

```python
def es_par(numero):
    """
    Determina si un número es par.

    Args:
        numero (int): Número a evaluar.

    Returns:
        bool: True si el número es par, False en caso contrario.
    """
    return numero % 2 == 0
```
